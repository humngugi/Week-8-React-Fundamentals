## Project Description
Built an interactive article platform about online mobile gaming using React fundamentals. The application showcases modern React concepts including components, JSX, props, state management with useState, event handling, and conditional rendering. The article explores the rise of mobile gaming, popular games, industry statistics, and future trends in the gaming world.

## Technologies Used
- **React 18** - UI library
- **Vite** - Build tool and development server
- **JavaScript (ES6+)** - Programming language
- **JSX** - JavaScript XML syntax
- **CSS3** - Styling with modern features
- **Git & GitHub** - Version control

## Features
- ✅ **Component-Based Architecture** - Modular Header, Footer, and Article components
- ✅ **Interactive Like System** - useState hook for managing likes with toggle functionality
- ✅ **Dynamic Content Rendering** - Props-based data passing for gaming statistics and popular games
- ✅ **Controlled Components** - Event handling for user interactions
- ✅ **Conditional Rendering** - Different UI states for liked/unliked articles
- ✅ **Responsive Design** - Mobile-first approach with breakpoints
- ✅ **Modern UI/UX** - Gradient backgrounds, animations, and hover effects
- ✅ **Gaming Statistics Display** - Interactive cards showing industry data
- ✅ **Popular Games Table** - Dynamic list rendering with map and keys
- ✅ **Professional Article Layout** - Clean typography and content sections
